# PlanAula

Aplicación web para la gestión y planificación de aulas y horarios en centros educativos.

PlanAula permite organizar asignaturas, espacios y planificación semanal de forma clara y estructurada, facilitando la administración académica y la visualización de horarios.

[Ver aplicación en producción](https://planaula.onrender.com)
---

## Descripción

PlanAula es una herramienta diseñada para optimizar la organización de recursos educativos como aulas, asignaturas y horarios.  
Está orientada a centros educativos que necesitan una planificación clara, editable y accesible desde la web.

Funcionalidades principales:

- Gestión de aulas y espacios
- Planificación semanal de horarios
- Organización de asignaturas y recursos
- Sistema de usuarios con inicio de sesión
- Visualización clara de la planificación

---

## Tecnologías utilizadas

- **Frontend:** Thymeleaf (templates server-side)
- **Backend:** Spring Boot (Java)  
- **Base de datos:** PostgreSQL / MySQL  
- **Control de versiones:** Git y GitHub  
- **Despliegue:** Render
